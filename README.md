# README

# Requirements

1. Ruby version => 3.0.2
2. Postgresql

# Set up

1. Clone `test-rails-api` repository
2. Run `bundle install`

# DB Setup

1. Update database.yml file
2. run `rails db:setup`
3. run `rails db:seed`

# Running of Test Scripts using Rspec

1. run `rspec`

# Run the Server

1. run `rails s`

# API Doc

1. go to `localhost:3000/apipie`
